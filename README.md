# Help Recrutamento

Plataforma digital de recrutamento desenvolvida para a Help Recrutamento,
empresa de RH parceira da Rumo Logística em Rondonópolis, MT.

## ✨ Funcionalidades

- Listagem de vagas com filtros por área
- Modal de candidatura com formulário completo
- Banco de talentos para cadastro livre
- Consentimento LGPD integrado
- Layout responsivo (mobile-first)

## 🛠️ Tecnologias

HTML · CSS · JavaScript · EmailJS
